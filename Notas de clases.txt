Clase 12/06/23 
Git es un software de control de versiones
Configurar Git:
configurar nombre: git config --global user.name "Nombre_de_usuario"
consultar nombre: git config user.name
Agregar mail: git config --global user.email "mail"
consultar email:  git config user.email
consultar mail y nombre: git config --list (ver en las ultimas lineas)

Manual de git: git help config 

Cada vez que se inicia un proyecto, hay que iniciar el software de control de versiones
1. Abrir una terminal
2. Abrir git bash en la terminal en la opcion +
3. Iniciar git en el proyecto: git init 
4. Saber el estado de nuestros archivos: git status
5. git add nombre_de_archivo (cargar un solo archivo) o git add . (carga todos los archivos)
